map.rbt.no

One link that opens the maps app — on any device.

The problem

There is no universal way to open a map natively across devices.

  • maps.apple.com — used to work everywhere. Apple removed cross-platform support.
  • maps.app.goo.gl — Google's short links. Shut down.
  • Android needs geo: intents, iOS needs deep links, desktop opens a browser.

The fix

One link. Every platform. Just add ?q= with a place name.

map.rbt.no/?q=Eiffel+Tower,+Paris

How it works

The link detects the visitor's platform and opens the right maps app.

Android → native maps app
iOS → Apple Maps
macOS → Apple Maps
Windows / Linux → Google Maps web
In-app → tries native, falls back to Google Maps web

Try it

Enter a place name or address and hit go.

No cookies — only essential technicalities and anonymized visitor stats.
