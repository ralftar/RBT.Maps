map.rbt.no

One link that opens maps — on any device.

What is this?

map.rbt.no is a tiny redirect service. Add a ?q= parameter with any place name or address, and it will open the right maps app on the visitor's device — Google Maps, Apple Maps, or the web.

map.rbt.no/?q=Eiffel+Tower,+Paris click to copy

How it works

The link detects the visitor's platform and redirects accordingly. No tracking, no cookies, no JavaScript frameworks — just a single lightweight HTML file.

Android → geo: intent
iOS → Apple Maps / Google Maps
Desktop → Google Maps web
In-app → open in browser hint

Try it

Enter a place name or address and hit go.
